# Specification

## Summary
**Goal:** Rebrand the app to "Shoe Swapper" with a red/black/white theme, swap-focused language, a new Negotiation tab, and a trading hero image on the home page.

**Planned changes:**
- Rename all branding references from the old app name to "Shoe Swapper" (header logo, HTML title, and throughout the UI)
- Update all CTAs and headings to use trade/swap language (e.g., "Start Trading", "Swap") instead of buy/sell language
- Retheme the entire app to a red, black, and white color palette — replacing neon/electric accent colors with bold red (#E11D48 or similar), black backgrounds, and white text
- Apply the new theme consistently across all pages (Home, Marketplace, CreateListing, Profile)
- Add a "Negotiation" link to the main navigation and create a new Negotiation page/route
- Negotiation page shows a list of mock trade conversations (left panel) and a message thread view (right panel), each referencing a shoe listing by name; users can type and send messages stored in local state
- Add a trading hero image to the Home page hero section, served from `frontend/public/assets/generated/trading-hero.dim_1200x600.png`

**User-visible outcome:** Users see a fully rebranded "Shoe Swapper" app in red, black, and white with swap-focused language, can navigate to a Negotiation page to browse and participate in mock trade conversations, and see a dynamic trading hero image on the home page.
